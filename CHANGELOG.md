# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Full xterm.js terminal with ANSI colors and interactive program support
- Multi-language support: Shell (bash/zsh/sh), Python, JavaScript, TypeScript
- Persistent interpreter sessions that preserve variables across code blocks
- Session isolation per note
- "Run All" command to execute entire runbooks sequentially
- Runme-compatible code block annotations
- Configurable settings: interpreter paths, font sizes, editor behavior
- Tab management for multiple terminal sessions

### Notes
- Desktop only (requires Node.js child processes)
- First public release
